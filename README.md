# LabSismic
Repository to store and manage the exercises from the subject: microprocessor systems laboratory (ENE0440 - LABORATÓRIO DE SISTEMAS MICROPROCESSADOS) of the University of Brasilia. According to the syllabus the subject approaches the following topics:

1. MSP430 Architecture
2. Instruction Set and Assembly Programming
3. Input/Output (GPIO)
4. C Programming
5. Timers
6. Interrupts
7. Clock Configuration
8. Serial Communication
9. ADC Converter with Direct Memory Access (DMA)

— Objectives of the course: To familiarize the student with microcontroller programming. Study of the main topics regarding programming for microcontrollers and embedded systems (timers, interrupts, synchronous and asynchronous serial communication, A/D conversion). To develop solutions for everyday problems using microcontrollers.
